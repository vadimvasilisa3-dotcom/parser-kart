# STYLE_LEDGER — реестр стилей агентства

Проверяется на ШАГ 0.5 перед фиксацией палитры для каждого нового клиента.
Правило: новая запись не может совпадать с существующей по двум и более
полям (Accent hex ±10% по тону, Display font, Signature-элемент, Layout-концепция).
