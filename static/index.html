Собирать для ИИ

В ZIP: PROMPT.md (данные) + AGENT_WEBSITE.md (инструкции агенту, палитра по фото). После ZIP файлы удаляются с сервера.

Фильтрация выгрузки

Оставить организации, у которых нет отмеченных полей. При нескольких галочках — режим «Хотя бы одно»: достаточно отсутствия одного пункта (есть сайт, но нет отзывов — попадёт).

Сбор по ссылке на карточку

Вставьте ссылку с Яндекс.Карт — в том числе из вкладки «Фото» (…/gallery/…). Соберётся до указанного лимита фото (не все 276 сразу). Используйте кнопку ниже, не «Начать сбор».
