# Превью собранных сайтов

Сайты лежат в `sites/<slug>/`. Фото и данные — в `orgs/`.

## Запуск (правильный способ)

```bash
py scripts/serve_preview.py
```

Откроется: **http://127.0.0.1:8777/sites/only-ya/**

## Не используйте порт 8765

На `8765` часто висит другой проект (Flask phish-sim) — оттуда 404 «Не найдено».

## Пути к фото в сайте

Агент должен копировать фото в `sites/<slug>/assets/photos/` **или** ссылаться на `../../orgs/NNN_Название/photos/` — но папка `orgs/…/photos/` должна существовать локально (скачайте ZIP из парсера).
