# Салон-Маркетолог — self-service для владельцев салонов

Заменяет куски работы SMM, таргетолога, локального маркетолога и аналитика. Владелец делает сам, приложение объясняет каждый шаг.

## Запуск (один клик)

```
Дважды нажмите ЗАПУСК.bat
```

Скрипт сам: установит зависимости, запустит сервер, откроет браузер.

Подробнее: `ИНСТРУКЦИЯ.txt`

## Модули

| Вкладка | Что даёт |
|---|---|
| Диагностика | Потери в ₽/мес + что чинить первым |
| SMM | Контент-план 14 дней, тексты постов |
| Таргет | Объявления ВК/Яндекс, прогноз заявок |
| Карты | Чеклисты Яндекс.Карты, 2ГИС, отзывы |
| Аналитика | 6 цифр недели → вердикт |
| Интеграции | Подключение ВК и Яндекс (опционально) |

## PDF

Кнопки «Скачать PDF» на вкладках Диагностика и Аналитика → в диалоге печати «Сохранить как PDF».

## Тесты

```
node test.js
cd backend && py -3 smoke_test.py
```

## Автоподключение ВК/Яндекс (опционально)

Нужны ключи в `backend/.env` (см. `.env.example`). Без ключей всё остальное работает.
