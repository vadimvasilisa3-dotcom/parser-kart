# Салон-Маркетолог — self-service приложение для владельцев салонов

Заменяет 5 специалистов: SMM, таргетолог, локальное продвижение, digital-маркетолог, performance. Владелец делает всё сам, приложение объясняет каждый шаг.

## Запуск

Двойной клик по `ОТКРЫТЬ.bat` или открыть `index.html` в браузере. Бэкенд не нужен, данные хранятся в localStorage браузера.

## Модули

| Вкладка | Роль специалиста | Что делает |
|---|---|---|
| Диагностика | Performance | Топ потерь в ₽/мес (no-show, загрузка, спящая база, пропущенные обращения) + маршрут «что чинить» |
| SMM и контент | SMM/контент-менеджер | Генератор контент-плана на 14 дней под нишу (тексты copy-paste + подсказки для сторис), чеклист оформления ВК |
| Таргет и реклама | Таргетолог | Прогноз заявок из бюджета по CPL ниши, готовые объявления ВК/Яндекс, генератор UTM |
| Карты и отзывы | Локальное продвижение | Чеклисты Яндекс.Карты (12) / 2ГИС (6) / отзывы (6) с сохранением прогресса, скрипты сбора отзывов |
| Аналитика | Performance | 6 цифр недели → CTR/CPL/конверсии + вердикты «что выключить, что масштабировать» |
| Интеграции | Digital | Пошаговые мастера: VK Ads, Яндекс.Бизнес, Директ, 2ГИС, виджет онлайн-записи |

## Ниши (6)

Маникюр, брови/ресницы, волосы, косметология, барбершоп, массаж/SPA. Каждая — свои бенчмарки CPL, средний чек, цикл процедуры, услуги, шаблоны.

## Ограничения v1 (честно)

- Интеграции — мастера настройки, не авто-API. Полный OAuth-запуск кампаний (VK Ads API, Яндекс.Директ API, YML-фид прайса в Яндекс.Бизнес) — v2, требует ключей владельца и бэкенда.
- Прогнозы — модель по отраслевым бенчмаркам, не гарантия.

## Roadmap v2

1. Бэкенд + OAuth: авто-создание кампаний VK Ads, авто-синхронизация прайса в Яндекс.Бизнес (YML).
2. Импорт статистики из кабинетов по API → аналитика без ручного ввода.
3. Интеграция с YClients/Dikidi: реальные no-show и база для диагностики.
4. Мультисалонный режим (для продажи как SaaS-коробки).
